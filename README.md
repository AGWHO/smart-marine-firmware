# Smart Marine Monitoring Prototype

This repository contains the firmware development for a Smart Marine Weather and GPS Monitoring System prototype.

The system is being developed using an Orange Pi platform and multiple sensors. Development follows a version-based integration approach where components are added incrementally.

## Development Versions

Version 1
- Orange Pi setup
- JSN proximity sensor integration

Version 2
- Add tilt sensor

Version 3
- Add GPS module

Version 4
- Add communication module

## Current Status
